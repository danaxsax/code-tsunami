# Code Tsunami

B2B mobile-first ordering platform for small convenience stores ("abarrotes") in Mexico. Built with React 19 and Vite 6.

## Features

- **Inicio** — Point-of-sale dashboard with store info, product search, restock suggestions, and cart subtotal.
- **Productos** — Product catalog organized by category (sodas, water, snacks, dairy, etc.).
- **Pedidos** — Order history with status tracking (delivered, in transit, canceled) and filters.
- **Gana** — Loyalty rewards program with points balance, challenges ("retos"), and a rewards store.
- **AI** — Chat assistant (in development, currently echoes a placeholder response).

## Tech Stack

- [React 19](https://react.dev/)
- [Vite 6](https://vitejs.dev/)
- CSS (custom properties, mobile-first design)

## Getting Started

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal (default `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```
